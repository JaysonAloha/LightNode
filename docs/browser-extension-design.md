# 轻知浏览器插件 - UI 设计稿

## 功能概述
用户在任何网页选中文字后，右键菜单可一键存入轻知知识库。

## 交互流程

1. **选中文本** → 用户用鼠标选中网页上的任意文字
2. **右键菜单** → 出现「存入轻知」选项
3. **点击** → 弹出轻量弹窗，可编辑/补充备注
4. **确认** → 内容同步至轻知（需登录）

## UI 原型

### 右键菜单项
```
┌─────────────────┐
│ 复制            │
│ 复制为纯文本    │
├─────────────────┤
│ 📥 存入轻知     │  ← 新增
└─────────────────┘
```

### 快速保存弹窗（点击后）
```
┌──────────────────────────────────────┐
│  存入轻知                      [×]   │
├──────────────────────────────────────┤
│  ┌────────────────────────────────┐ │
│  │ [选中的原文，可编辑]            │ │
│  │                                │ │
│  └────────────────────────────────┘ │
│  来源：https://example.com/article  │
│  ┌────────────────────────────────┐ │
│  │ 备注（可选）                    │ │
│  └────────────────────────────────┘ │
│                                      │
│         [取消]    [保存到轻知]       │
└──────────────────────────────────────┘
```

### 未登录状态
```
┌──────────────────────────────────────┐
│  存入轻知                      [×]   │
├──────────────────────────────────────┤
│  请先登录轻知以同步内容              │
│  [打开轻知并登录]                    │
└──────────────────────────────────────┘
```

## 技术要点
- Manifest V3
- chrome.contextMenus / browser.contextMenus
- 与主站通信：localStorage 或 API（需 CORS）
- 可选：Popup 常驻入口，点击打开主站
